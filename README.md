# novah-ide README

IDE for the [Novah](https://github.com/stackoverflow/novah) programming language.

## Features

- Syntax highlighting
- Snippets
- Code completion
- Go to definition
- Semantic coloring
- Hover
- Formatting
- Folding

More to come.

## Requirements

- Java installed and JAVA_HOME set.
- Novah compiler jar.
