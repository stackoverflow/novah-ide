# novah-ide README

IDE for the [Novah](https://github.com/stackoverflow/novah) programming language.

## Features

- Syntax highlighting
- Snippets
- Code completion
- Go to definition
- Semantic coloring
- Hover
- Formatting
- Folding
- Find all references
- Rename symbol (variable, declaration, type or constructor)

More to come.

## Requirements

- Java installed and in the path or JAVA_HOME set.
- Novah compiler jar.
